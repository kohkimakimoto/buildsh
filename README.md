# buildsh

A docker image and shell script for building testing and deploying softwares.

WIP...

## Installation

```
$ git clone https://github.com/kohkimakimoto/buildsh.git ~/.buildsh
$ echo 'export PATH="$HOME/.buildsh/bin:$PATH"' >> ~/.bash_profile
```
